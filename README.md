# ARIF DSA Challenges

🚀 Welcome to the Data Structures & Algorithms challenge repository of the AI Research and Innovation Forum (ARIF)!

## 📌 About
This repository hosts weekly DSA problems curated to help students strengthen their problem-solving and coding skills. You can solve them in any programming language of your choice.

## 🛠 How to Contribute
1. Fork the repo
2. Clone it and create a new branch:  
   `git checkout -b feature/your-solution-name`
3. Add your solution under the correct week folder
4. Commit and push changes
5. Create a Pull Request

## ✅ Rules
- Add comments in your code
- Mention your language and name at the top
- One file per problem per contributor
